# practice

* Install composer 
  - https://getcomposer.org/
* Install laravel 
``` 
composer global require "laravel/installer" 
```
* Navigate in terminal to projectfolder then install npm
```
npm install
```
* go to projectfolder (more database info -> https://laravel.com/docs/5.6/migrations)
```
php artisan migrate
```

## Run program
* 1st - run 
```
php artisan serve
```
* 2nd - run 
```
npm run watch
```
